   1. wget
      * 用于从网络上下载资源。
   2. tar 
      * Linux下的归档使用工具,用来打包和备份。
   3. mv 
      * 用来对文件或目录重新命名   
  4. ln
      * 用来为文件创建连接 <ln -s /usr/local/node-v6/bin/node /bin/node>
      1. -s --symbolic  创建相对于链接位置的符号链接     
  5. echo
     * 输出指定的字符串或者变量 
  6. source
     * 在当前Shell环境中从指定文件读取和执行命令,命令返回退出状态。
