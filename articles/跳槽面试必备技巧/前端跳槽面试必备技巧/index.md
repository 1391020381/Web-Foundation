# 安排
1. 面试准备
    * JD描述分析
    * 业务分析
    * 技术栈准备

2. 模拟一面
    * 页面布局类
    * CSS和模型 DOM事件类
    * HTTP协议类  原型链类
    * 面向对象类  、通信类
    * 前端安全类  前端算法类   
3. 模拟二面
    * 渲染机制类
    * JS运行机制
    * 页面性能
    * 错误监控 
4. 模拟三面
    * 业务能力
    * 团队协作能力
    * 带人能力
5. 模拟终面
    * 职业竞争力
    * 职业规划
