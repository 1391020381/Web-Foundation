# 本书的约定:
 1. 代码以等宽字体表示。
 2. 新加的或修改过的代码通常会以加粗的等宽子弟表示。
 3. 伪代码和变量输入以斜体等宽字体表示。
 4. 对于需要提醒读者注意的内容,突出排版。
 5. 有时某一行代码可能会超出本书的宽度,这时将使用箭头图标
 6. 为了节省版面，许多例子代码中都会包含'...省略的代码....'
 7. 在要读者按照作者的指示构建例子的情况下,会以一个带注释的结构开头。
