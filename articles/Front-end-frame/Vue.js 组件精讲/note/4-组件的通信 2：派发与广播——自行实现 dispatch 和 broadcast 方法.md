- 子组件向父组件(支持跨级)传递数据

# Vue.js 1.x 的 $dispatch 与 $broadcast

- $dispatch 用于向上级派发事件,只要是它的父级(一级或多级以上),都可以在组件内通过 $on 监听到

* \$broadcast 由上级向下级广播事件的。
