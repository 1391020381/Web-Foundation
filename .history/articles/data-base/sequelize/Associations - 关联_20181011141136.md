* Sequelize中有四种类型的关联。
1. BelongsTo
2. HasOne
3. HasMay
4. BelongsToMany
* Source & Target
```
const User = sequelize.define('User',{
    name:Sequelize.STRING,
    email:Sequelize.STRING
});
const Project = sequelize.define('Project',{
    name:Sequelize.STRING
});

User.hasOne(Project)

User模型(函数被调用的模型)是source。Project模型(作为参数传递的模型)是target
```
